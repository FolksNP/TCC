Instalando o SASS
Para instalar o SASS é nescessário digitar no terminal o seguinte código:
código => npm install -g sass

depois digite o seguinte código para verificar se o sass foi Instalando
código => sass --version

após esse processo, agora vamos fazer o arquivo scss compilar

em cima da pasta styles, clique com o botão direito, onde irá abrir uma telinha com algumas opções;
seleciona a seguinte opção => Open in Integrated Terminal

após isso digite o seguinte comando:

sass --watch sass/main.scss css/main.css

após isso o sass deve estar compilando o arquivo.

lembrando que a partir de agora, iremos compilar apenas o arquivo main.scss, e nenhum outro arquivo.
